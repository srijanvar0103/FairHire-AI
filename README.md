Project Description

FairHire AI is an intelligent web-based recruitment system designed to eliminate bias in hiring by evaluating candidates purely based on their skills and experience. The system processes PDF resumes, extracts relevant information, and assigns a transparent score to ensure fair decision-making.

Inspired by modern AI-driven solutions like AutoTalker, which leverage AI to generate personalized content and enhance user experiences, FairHire AI applies similar principles to recruitment by automating evaluation and reducing human bias.

🔍 Problem Statement

In traditional hiring systems, decisions are often influenced by unconscious biases such as:

Gender and personal identity
Educational background or institution
Resume format and presentation

These biases result in:

Unfair candidate rejection
Lack of diversity in hiring
Missed opportunities for skilled individuals
💡 Solution

FairHire AI provides a structured and unbiased evaluation process by:

Extracting text from PDF resumes
Removing bias-related indicators
Identifying relevant technical skills
Generating a transparent candidate score
Providing clear shortlisting decisions
⚙️ Approach

The system uses a combination of simple NLP techniques and rule-based evaluation to simulate an AI-driven hiring process:

Resume Upload (PDF)
Text Extraction using PyPDF2
Bias Removal (filtering personal indicators)
Skill Detection using keyword matching
Score Calculation
Decision Output (Shortlisted / Rejected)



🧠 Key Features

   📄 PDF Resume Upload & Processing
   
   🧹 Bias Removal Simulation
   
   🧠 Skill-Based Evaluation
   
   📊 Transparent Scoring System
   
   ✅ Shortlisting Recommendation
   
   🎨 Modern Web Interface (Flask-based)



🛠️ Tech Stack 

    Backend: Python, Flask

    Frontend: HTML, CSS
    
    Logic: Basic NLP / Rule-based AI
